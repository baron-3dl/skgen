self-building skills

Skills that don't break
when you share them.

Your AI skill is dialed in. You hand it to someone and it falls apart, because it quietly hooked into your setup. skillc is the tool that ships it as one file that rebuilds itself on whoever opens it, then proves it matched your quality or says straight up that it couldn't.

Install skillc in about 10 seconds

Paste into Claude Code. That installs the builder. Then: build a self-building skill from <your skill>

1
Turn on code execution. Settings, then Capabilities.
2
Download the builder (one tap):
skillc builder ↓
3
Upload it. Settings, then Capabilities, then Skills, then drop the zip in. Now ask it to build a skill from one you have.

how it works

It carries its own builder.

Like a self-extracting zip, the file holds the contents and the unpacker. The first time someone uses it, before it answers anything, it builds itself against their machine.

01

Rebuilds on their model

It re-derives the instructions your skill needs on their model and their setup, overriding any local style that fights it.

02

Tests itself

It scores against examples you approved that it never trained on. That is the honest "did it transfer" number, not a memory check.

03

Tells the truth

One line: built, honest-failure with the gap, or cannot-build with the missing tool named. No silent drift, ever.

04

You approve, it ships

Authoring is one manual step: you bless the good outputs your skill already makes. skillc does the rest and hands you one file.
